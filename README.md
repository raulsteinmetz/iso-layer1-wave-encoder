# Wave Encoder

Wave Encoder is a Python-based interface that specializes in encoding bit sequences into waveforms.

### Dependencies

`sudo apt-get install python-pip`

`pip3 install numpy`

`pip3 install matplotlib`

`pip3 install pygame`

### Running interface
`python3 main.py`
